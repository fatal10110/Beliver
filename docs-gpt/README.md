# Game Work Breakdown (High Level)

This folder contains a high-level breakdown of the work needed to realize the game described in `concept.md`.

## Index

- `business.md` — market, monetization, legal/compliance, community, operations
- `economy.md` — resources, sinks/sources, progression, balance, seasons, monetization boundaries
- `game_design.md` — core loop, systems, UX flows, multiplayer rules, content scope
- `tech.md` — architecture, backend/services, client, data, security, tooling, build/release
- `content.md` — narrative, faction content, art/audio, localization, sensitivity review
- `production.md` — team, milestones, deliverables, risks, estimates
