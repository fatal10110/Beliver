# Production Roadmap

## 1. Timeline (Estimated 18 Months)

### Phase 0: Pre-Production (Months 1-3)
*   **Goal**: Vertical Slice / POC.
*   **Deliverables**:
    *   Playable Web POC (Hex grid, basic move/attack).
    *   Finalized Art Style Guide.
    *   Tech Stack selected.

### Phase 1: Foundation (Months 4-9)
*   **Goal**: Alpha Build.
*   **Deliverables**:
    *   Unity Client basic implementation.
    *   Backend Multiplayer working (Lobby + Sync).
    *   1 Faction fully playable.

### Phase 2: Production (Months 10-15)
*   **Goal**: Beta Build.
*   **Deliverables**:
    *   All 3 Factions complete.
    *   Campaign (Arc 1) complete.
    *   Economy/Shop integration.
    *   Closed Beta testing commences.

### Phase 3: Polish & Launch (Months 16-18)
*   **Goal**: Global Release.
*   **Deliverables**:
    *   Bug fixing & Optimization.
    *   Localization.
    *   Marketing campaign ramp-up.
    *   Launch Day!

## 2. Team Structure (Ideal Sizing)
*   **Production**: 1 Producer.
*   **Design**: 1 Lead Designer.
*   **Engineering**:
    *   1 Backend Lead (Go).
    *   1 Gameplay Engineer (Unity).
    *   1 Frontend/UI Engineer.
*   **Art**:
    *   1 3D Generalist (Environment/Units).
    *   1 UI/2D Artist.
*   **QA**: 1 QA Lead (Contractors later).

## 3. Tools
*   **Project Management**: Jira or Linear.
*   **Communication**: Slack/Discord.
*   **Version Control**: Git (GitHub/GitLab).
*   **CI/CD**: GitHub Actions.
